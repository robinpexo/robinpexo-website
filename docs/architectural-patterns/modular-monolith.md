## Overview

A modular monolith keeps clear module boundaries while staying deployable as one unit.

## When it helps

- Early stage teams
- Systems that need strong transactional boundaries

## Module boundary sketch

```text
API
 ├─ Catalog
 ├─ Orders
 └─ Billing
```
